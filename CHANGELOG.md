# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Conversation history tracking
- Personalized responses with parameter substitution
- 4-level triage system (Emergency, Urgent, Same-Week, Routine)
- Web frontend with Flask backend
- Comprehensive documentation

### Fixed
- Parameter substitution in messages
- Duplicate message issues
- Duration parameter extraction
- Triage parameter logic
- Message formatting and flow

### Changed
- Improved conversation flow
- Enhanced triage logic with better conditions
- Cleaner, more professional responses
- Better error handling

## [0.1.0] - 2025-11-05

### Added
- Initial Dialogflow CX agent setup
- Symptom intake flow
- Clarifying questions page
- Triage evaluation
- Summary and recommendations
- Basic frontend interface
- Test scenarios and documentation

### Known Issues
- RAG integration requires manual datastore setup
- Some edge cases in parameter extraction may need refinement
